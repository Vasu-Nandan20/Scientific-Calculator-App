# Numix - Premium Scientific Calculator

A professional, feature-rich Scientific Calculator app for Android built with modern technologies.

## 🚀 Features
- **Smart Scientific Mode**: Advanced mathematical functions (Trig, Logs, Powers, Roots, Factorials) with an animated expandable panel.
- **Dynamic Function Plotter**: Interactive graphing area with pinch-to-zoom and panning support.
- **Ultimate Unit Converter**: 10+ categories including Length, Weight, Temperature, Area, Volume, Speed, Pressure, Power, and Number System (Base) conversion.
- **Static Conversion Logic**: All conversions are performed locally with high precision—no internet or APIs required.
- **Professional UI/UX**: Modern Glassmorphism/Material You inspired design with smooth animations and haptic feedback.
- **History Tracking**: Keep track of your previous calculations and reuse them with a single tap.

## 🛠️ Tech Stack
- **Language**: Kotlin
- **Framework**: Jetpack Compose
- **Architecture**: MVVM (Clean Architecture)
- **Math Engine**: mXparser
- **Navigation**: Jetpack Navigation Compose

## 📦 Setup & Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/Vasu-Nandan20/Scientific-Calculator-App.git
   ```
2. Open the project in **Android Studio (Ladybug or newer)**.
3. Sync the Gradle project.
4. Run the app on an emulator or physical device.
